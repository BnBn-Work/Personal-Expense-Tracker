# Personal-Expense-Tracker
An app designed to allow you to conviently track all your income, expenses and budgeting in one place.

# Setup
To run the server you must have [Node Js](https://nodejs.org/en/download) and [MySQL](https://dev.mysql.com/downloads/installer/) installed.
Then, make a local clone of this repo & open a terminal in that directory.
Finally, add config.json to the root of the project & add the property "databasePassword" with the value as your root password to mySQL as a string.

# Use
To launch the backend, simply run the command `node server.js`.
The front end can be reached by accessing port 2000 on the computer that is hosting the backend through a browser.
For example, http://localhost:2000 if running locally.
